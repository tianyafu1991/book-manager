操作提示层

提交中提示（该提示不会自动消失，调用成功或失败时会自动消失，亦可使用javaex.close();来手动关闭）
<button id="demo1" class="button blue">提交中提示</button>

<script>
	$("#demo1").click(function() {
		javaex.tip({
			content : "提交中...",
			type : "loading"
		});
	});
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 content 加载说明
2 type 弹出层类型

提示层代码
<button id="demo2" class="button blue">提示层</button>

<script>
	$("#demo2").click(function() {
		javaex.tip({
			content : "我是提示层"
		});
	});
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 content 提示内容
2 live 2000 多少毫秒后消失。1000毫秒=1秒
3 marginTop 距离顶部的距离（可以是20%或20px之类的）如果你不想居中，可以设置该值
4 marginBottom 距离底部的距离（可以是20%或20px之类的）如果你不想居中，可以设置该值
5 mask true 是否有遮罩，没有时，可以点击页面的元素（例如，登录页面的校验，你可以取消遮罩，并让该提示层在底部提示，从而让用户可以无需等待再次输入内容）
