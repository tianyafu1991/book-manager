高级弹出层（菜单）

代码
<button id="demo" class="button blue">菜单样式</button>

<script>
	$("#demo").click(function() {
		javaex.dialog({
			type : "menu",
			menu : {
				// 允许加图标等样式，但是最外层必须用单引号包裹
				'<span class="icon-trash"></span> 删除' : "del()",
				'<span class="icon-block"></span> 禁用' : "disable()"
			}
		});
	});

	function del() {
		javaex.confirm({
			content : "确定要删除么？",
			callback : "callback()"
		});
	}
	
	function callback() {
		alert("点击了确定");
	}
	
	function disable() {
		alert("点击了禁用");
	}
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 type 弹出层类型。必须填 menu
2 menu json字符串。写法请参照示例，菜单名对应的函数留空，视为关闭，不做任何操作
