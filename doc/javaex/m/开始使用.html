开始使用

首先，创建一个手机网页文档
<!DOCTYPE html PUBLIC "-//WAPFORUM//DTD XHTML Mobile 1.0//EN" "http://www.wapforum.org/DTD/xhtml-mobile10.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>无标题文档</title>
</head>

<body>
</body>
</html>

然后，你需要加载以下文件
<!--字体图标-->
<link href="javaex/m/css/icomoon.css" rel="stylesheet" />
<!--动画-->
<link href="javaex/m/css/animate.css" rel="stylesheet" />
<!--骨架样式-->
<link href="javaex/m/css/common.css" rel="stylesheet" />
<!--皮肤-->
<link href="javaex/m/css/skin/default.css" rel="stylesheet" />
<!--jquery-->
<script src="javaex/m/lib/jquery-3.3.1.min.js"></script>
<!--手机适配-->
<script src="javaex/m/js/flexible.js"></script>
<!--全局动态修改-->
<script src="javaex/m/js/common.js"></script>
<!--核心组件-->
<script src="javaex/m/js/javaex.min.js"></script>
<!--表单验证-->
<script src="javaex/m/js/javaex-formVerify.js"></script>
