日期时间

可以使用div或span等其他元素来承载日期组件
如果你想用框架内置的表单验证插件在表单中对其进行验证。则必须使用input元素来承载，并且将初始化日期设置为空（缺省date属性）。
代码
<input id="date" type="text" placeholder="点击选择" value="" readonly />

<script>
	javaex.date({
		id : "date",
		// date : "2018-01-01",	// 默认选择的日期
		// 重新选择日期之后返回一个时间对象
		callback: function (rtn) {
			alert(rtn.date);
		}
	});
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 id 承载日期组件的id
2 date 默认选择的日期
3 callback 重新选择日期之后返回一个时间对象。必须按照示例上的写
