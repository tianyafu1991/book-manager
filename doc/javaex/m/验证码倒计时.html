验证码倒计时

获取验证码
代码
<a href="javascript:;" style="color: #0087e0;" id="get_code" onclick="getCode()">获取验证码</a>

<script>
	function getCode() {
		javaex.code({
			id : "get_code",
			color : "#0087e0",
			downColor : "#0087e0"
		});
	}
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 id 点击触发的元素id
2 second 45 倒计时秒数
3 text 获取验证码 倒计时结束后，重新显示的文字
4 color unset 倒计时结束后，重新显示的文字颜色
5 backgroundColor transparent 倒计时结束后，重新显示的文字背景色
6 downColor unset 倒计时时，显示的文字颜色
7 downBackgroundColor transparent 倒计时时，显示的文字背景色
8 type 设为reset时，可以手动重置
