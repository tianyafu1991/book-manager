普通弹出层

确认框代码
<button id="demo1" class="button blue">确认框</button>

<script>
	$("#demo1").click(function() {
		javaex.alert({
			content : "我是来测试的",
			callback : "callback()"
		});
	});

	function callback() {
		alert("我是回调函数");
		// 如果你想阻止弹出层关闭，直接 return false; 即可
		//return false;
	}
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 content 弹出层内容
2 confirmName 确定 确定按钮名称
3 callback 点击确定按钮后的回调函数

确认选择框代码
<button id="demo2" class="button blue">确认选择框</button>

	<script>
		$("#demo2").click(function() {
			javaex.confirm({
				content : "确定要删除么？",
				callback : "callback()"
			});
		});

		function callback() {
			alert("我是回调函数");
			// 如果你想阻止弹出层关闭，直接 return false; 即可
			//return false;
		}
	</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 content 弹出层内容
2 confirmName 确定 确定按钮名称
3 cancelName 取消 取消按钮名称
4 callback 点击确定按钮后的回调函数
