select选择框

按照原生的方法使用即可，但是不再支持onchange事件，需要使用回调函数来代替onchange
代码
<select id="select">
	<option value="">请选择</option>
	<option value="1">大专</option>
	<option value="2">本科</option>
	<option value="3">硕士</option>
	<option value="4">博士</option>
	<option value="5">博士后</option>
</select>

<script>
	javaex.select({
		id : "select",
		// 选择后的回调函数，初始化时也会自动调用一次
		callback: function (rtn) {
			console.log("selectValue:" + rtn.selectValue);
			console.log("selectName:" + rtn.selectName);
		}
	});
</script>
参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 id select选择框的id
2 callback 选择后的回调函数，初始化时也会自动调用一次
使用该组件后，原生的onchange事件将会失效，如果你想做联动数据，则必须在此回调函数中进行
如果不需要联动修改其他的select框的数据，则可以不需要回调函数，可以用原生的方法获取值
