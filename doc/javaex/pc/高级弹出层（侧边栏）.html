<button class="javaex-btn blue" onclick="test1()">点我弹窗</button>

<script>
	function test1() {
		javaex.drawer({
			id : "drawer",
			width : "500px",
			isClickMaskClose : true,
			position : "right",
			callback : function() {
				console.log("弹窗已打开");
			}
		});
	}
</script>

使用方法一：iframe加载另一个页面
代码
<button class="javaex-btn blue" onclick="test2()">点我弹窗</button>

<script>
	function test2() {
		javaex.drawer({
			width : "500px",
			title : "我是标题",
			url : "https://www.baidu.com/",
			isClickMaskClose : true,
			position : "right",
			close : function() {
				console.log("点x关闭时产生的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	}
</script>
参数说明
参数 默认值 说明 版本
id 自定义页面必须指定id，iframe嵌套可以不指定
width 300 弹出层宽度。支持： 500、500px、30% 这3种写法
position right 弹出位置，可选值：left、right
isClickMaskClose false 是否允许点击遮罩关闭
title 温馨提示 弹出层标题，自定义页面时不用填该选项
closeIcon true 是否显示右上角关闭 X，自定义页面时不用填该选项
url 使用iframe嵌套时需要该参数
close 点击右上角X关闭弹出层时的回调函数。仅在iframe嵌套时有效
callback 打开弹窗后的回调函数
callback : function() {
	console.log("弹窗已打开");
}
