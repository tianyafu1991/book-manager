按钮

默认按钮
代码
<button class="javaex-btn blue">蓝色按钮</button>
<button class="javaex-btn navy">靛蓝色按钮</button>
<button class="javaex-btn indigo">靛青色按钮</button>
<button class="javaex-btn wathet">浅蓝色按钮</button>
<button class="javaex-btn red">红色按钮</button>
<button class="javaex-btn green">绿色按钮</button>
<button class="javaex-btn yellow">黄色按钮</button>
<button class="javaex-btn gray">灰色按钮</button>
<button class="javaex-btn disabled" disabled="disabled">禁用按钮（不可点击）</button>

镂空按钮：追加 class="empty"
代码
<button class="javaex-btn blue empty">蓝色按钮</button>
<button class="javaex-btn navy empty">靛蓝色按钮</button>
<button class="javaex-btn indigo empty">靛青色按钮</button>
<button class="javaex-btn red empty">红色按钮</button>
<button class="javaex-btn green empty">绿色按钮</button>
<button class="javaex-btn yellow empty">黄色按钮</button>
<button class="javaex-btn gray empty">灰色按钮</button>
<button class="javaex-btn disabled empty" disabled="disabled">禁用按钮（不可点击）</button>

迷你按钮（添加class="mini"即可）


代码
<button class="javaex-btn blue mini">蓝色按钮</button>
<button class="javaex-btn blue empty mini">蓝色按钮</button>

带图标的按钮
代码
<!--为了防止按钮文字和图标紧紧贴着，请在文字前敲一个空格-->
<button class="javaex-btn blue"><span class="icon-plus"></span> 图标按钮</button>
<button class="javaex-btn red"><span class="icon-minus"></span> 图标按钮</button>

圆角按钮
代码
<!--默认提供 3、5、8 的圆角-->
<button class="javaex-btn blue radius-3">圆角按钮</button>
<button class="javaex-btn blue radius-5">圆角按钮</button>
<button class="javaex-btn blue radius-8">圆角按钮</button>
按钮组
代码
<span class="javaex-group-button">
	<button class="javaex-btn blue"><span class="icon-plus"></span> 添加</button>
	<button class="javaex-btn red"><span class="icon-close2"></span> 删除</button>
	<button class="javaex-btn green"><span class="icon-check2"></span> 保存</button>
</span>
