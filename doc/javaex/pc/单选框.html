伪单选框（一定要在外面包一层label）
代码
<!--在label标签上添加class即可（选中和未选中的class不要加）-->
<label class="javaex-btn"><input type="radio" class="javaex-fake" name="radio3" value="1" checked/>入门型</label>
<label class="javaex-btn"><input type="radio" class="javaex-fake" name="radio3" value="2"/>标准型</label>
<label class="javaex-btn"><input type="radio" class="javaex-fake" name="radio3" value="3"/>企业型</label>
<label class="javaex-btn"><input type="radio" class="javaex-fake" name="radio3" value="4"/>豪华型</label>

<script>
	javaex.fakeRadio({
		name : "radio3",
		uncheckClass : "wathet", // 未选中时添加的class
		checkedClass : "blue", 	// 选中时添加的class
		isInit : true,			// 初始化是否调用回调函数
		callback : function(rtn) {
			console.log(rtn.val);
			console.log(rtn.text);
		}
	});
	
	// jquery原生方法取值
	$(':radio[name="radio3"]').change(function() {
		console.log($(this).val());
	});
</script>
参数说明
参数 默认值 说明 版本
name radio的name属性
uncheckClass 未选中时往label标签添加的class
checkedClass 选中时往label标签添加的class
isInit false 初始化是否自动调用回调函数
callback 返回选中的值和文本
