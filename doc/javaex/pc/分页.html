分页

    代码
    <div class="javaex-page">
	<ul id="page" class="javaex-pagination"></ul>
</div>

<script>
	javaex.page({
		id : "page",    // jquery选择器，指定到ul节点
		pageNum : 1,           // 默认选中第几页
		pageSize : 10,         // 每页显示多少条
		totalPages : 12,       // 总页数
		isShowJumpPage : true, // 是否显示跳页
		totalNum : 125,        // 总条数，不填时，不显示
		position : "left",
		callback : function(rtn) {
			console.log("当前选中的页数：" + rtn.pageNum);
			console.log("每页显示条数：" + rtn.pageSize);
		}
	});
</script>

    参数说明
    参数 默认值 说明 版本
    id 分页id
    pageNum 1 默认选中第几页
    pageSize 每页显示多少条
    totalPages 总页数
    totalNum 总条数，不填时，不显示
    isShowJumpPage true 是否显示跳页
    isShowSelect true 是否显示条数选择控件
    isShowOnePage true 当只有一页时，是否显示分页组件
    lastPageText 上一页 上一页文字标识
    nextPageText 下一页 下一页文字标识
    position right 分页显示位置。left:居左；center:居中；right:居右
    isReturnFirst true 条数选择控件重新选择后，是否返回第1页
    callback 返回一个分页对象。pageNum：当前选中的页数； pageSize：每页显示条数
