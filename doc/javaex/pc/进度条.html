进度条

代码
<div class="javaex-progress">
	<div id="progress1" class="javaex-progress-level"></div>
</div>

<!--如果要显示百分比，则需要自己为 class="javaex-progress" 设定 line-height-->
<div class="javaex-progress" style="line-height: 16px;margin-top: 50px;">
	<div id="progress2" class="javaex-progress-level"></div>
</div>

<script>
	javaex.progress({
		id : "progress1",
		percent : "40"
	});
	
	javaex.progress({
		id : "progress2",
		percent : "40",
		isShowPercent : true
	});
</script>
参数说明
参数 默认值 说明 版本
id 指定操作哪个进度条
percent 0 进度条百分比，例如：40 或 "40" 或 "40%"
isShowPercent false 是否显示百分比文字（如果要显示百分比，则需要自己为 class="progress" 设定 line-height）
