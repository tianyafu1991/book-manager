操作提示层

提交中提示（ajax提交前调用。只能依靠ajax返回数据后，再次调用成功或异常提示，才能让其消失，请无视演示中的自动消失）

该提示存在时，页面将无法再点击其他元素（防止用户重复提交）

适用于后台
代码
<script>
	$("#submit").click(function() {
		javaex.tip({
			content : "数据提交中，请稍候...",
			type : "submit"
		});
	});

	$("#success").click(function() {
		javaex.tip({
			content : "操作成功",
			type : "success"
		});
		// 建议延迟加载
		setTimeout(function() {
			// 刷新页面
			// window.location.reload();
			// 跳转页面
			// window.location.href = "${pageContext.request.contextPath}/zone_info/list.action";
		}, 1000);
	});

	$("#error").click(function() {
		javaex.tip({
			content : "操作失败",
			type : "error"
		});
	});
</script>

适用于前台门户
代码
<script>
	$("#submit2").click(function() {
		javaex.tip({
			mode : "message",
			content : "数据提交中，请稍候...",
			type : "submit"
		});
	});
	
	$("#success2").click(function() {
		javaex.tip({
			mode : "message",
			content : "操作成功",
			type : "success"
		});
	});
	
	$("#error2").click(function() {
		javaex.tip({
			mode : "message",
			content : "操作失败",
			type : "error"
		});
	});
</script>

适用于前台门户（黑色半透明）
代码
<script>
	$("#toast").click(function() {
		javaex.tip({
			mode : "toast",
			content : "请先阅读并同意协议"
		});
	});
</script>

参数说明
参数 默认值 说明 版本
mode 模式，为 message 时中间弹出，为空时是默认样式。后台使用时建议为空
content 弹出层内容
type success submit:提交中。 success:成功。 error:失败
live 2000 提示存在时间，单位:毫秒 （仅在type为 success 或 error 时有效）
timeout 10000（10秒） 提交超时时间（单位:毫秒），填0时，不会有超时提示
timeoutText 连接超时，请重试 超时提示
