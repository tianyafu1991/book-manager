消息反馈层

提交中提示（ajax提交前调用。只能依靠ajax返回数据后，再次调用成功或异常提示，才能让其消失，请无视演示中的自动消失）
该提示存在时，页面将无法再点击其他元素（防止用户重复提交）
代码
<button id="submit" class="button blue">提交中提示</button>
<script>
	$("#submit").click(function() {
		javaex.message({
			content : "数据提交中，请稍候...",
			type : "submit"
		});
	});
</script>

成功提示
代码
<button id="success" class="button blue">成功提示</button>

<script>
	$("#success").click(function() {
		javaex.message({
			content : "操作成功",
			type : "success"
		});
	});
</script>

异常提示
代码
<button id="error" class="button blue">异常提示</button>

<script>
	$("#error").click(function() {
		javaex.message({
			content : "操作失败",
			type : "error"
		});
	});
</script>

参数说明（使用属性默认值时，可以缺省该属性）
序号 属性名 默认值 说明
1 content 弹出层内容
2 type success success：成功。 error：失败
3 live 2000 提示存在时间，单位：毫秒。1000毫秒=1秒 （仅在type为 success 或 error 时有效）
4 timeout 0 提交超时时间（单位：毫秒），填0时，不会有超时提示
5 timeoutText 连接超时，请重试 超时提示
