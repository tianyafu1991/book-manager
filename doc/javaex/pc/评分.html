评分

整星演示
代码
<div id="score" class="javaex-score"></div>

<script>
	javaex.score({
		id : "score",
		callback: function (rtn) {
			alert("选中了第"+rtn.index+"个星星；分数为："+rtn.score);
		}
	});
</script>

半星演示

回显时，小数位 <0.7 时，会显示半星，否则显示全星

代码
<div id="score2" class="javaex-score"></div>

<script>
	javaex.score({
		id : "score2",
		num : 5,
		score : 4.6,
		scoreArr : [1, 2, 3, 4, 5],    // 即便开启了半星模式，也只要整数部分
		levelTextArr : ['0.5分', '1分', '1.5分', '2分', '2.5分', '3分', '3.5分', '4分', '4.5分', '5分'],
		half : true,
		clickOnce : true,
		callback: function (rtn) {
			alert("选中了第"+rtn.index+"个星星；分数为："+rtn.score);
		}
	});
</script>
参数说明
参数 默认值 说明 版本
id 评分组件div容器的id
num 5 星星数量
size 24 星星大小
scoreArr [1, 2, 3, 4, 5] 每个星星代表的分数，数组形式。
即便开启了半星模式，也只要整数部分
levelTextArr ['1分', '2分', '3分', '4分', '5分'] 每个星星的文字提示，数组形式。
如果开启了半星模式，则元素个数要是 scoreArr 的2倍
score 默认分数
isReadOnly false 是否只读
half false 是否启用半星
clickOnce true 是否只允许点击一次，点击后会自动变成只读模式
callback 返回一个评分对象，包含了第几个星星（index），和对应的分值（score）
