普通弹出层

确认弹出层
代码
<button id="demo1" class="javaex-btn blue">点我弹窗</button>

<script>
	$("#demo1").click(function() {
		var id = "xxx";
		javaex.alert({
			content : "我是来测试的",
			confirm : function() {
				console.log("获取变量值：" + id);
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});
</script>
确定取消弹出层
代码
<button id="demo2" class="javaex-btn blue">点我弹窗</button>

<script>
	$("#demo2").click(function() {
		javaex.confirm({
			content : "确定要删除么",
			confirm : function() {
				console.log("点击了确定按钮");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				return false;
			}
		});
	});
</script>
参数说明
参数 默认值 说明 版本
mask true 是否使用遮罩。如果你想无限弹窗，必须将mask设置为false
id 随机 如果页面同时存在多个弹层，可以自己指定id，并调用关闭弹层函数关闭指定弹层
width 300 弹出层宽度
maxHeight 弹出层内容区域最大高度，内容超出时，会显示滚动条
top 30% 距离顶部距离，支持 200px 这种写法。人的视线其实是要高于电脑屏幕的水平线的，所以这边没有设置成居中
title 温馨提示 弹出层标题
closeIcon true 是否显示右上角关闭 X
content 弹出层内容。可以是html代码
textAlign left 内容位置。left：居左，center：居中，right：居右
confirmName 确定 确定按钮名称
cancelName 取消 取消按钮名称
confirm 点击确定按钮后的回调函数
cancel 点击取消按钮后的回调函数
close 点击右上角X关闭弹出层时的回调函数
callback 打开弹窗后的回调函数
callback : function() {
	console.log("弹窗已打开");
}
