操作提示层-定位

代码
<button class="javaex-btn green ml-20 mt-20" onclick="toast1(this)">成功</button>
<button class="javaex-btn red ml-20 mt-20" onclick="toast2(this)">失败</button>

<script>
	// obj为当前点击的元素对象，obj是必须的
	function toast1(obj) {
		javaex.toast(obj, {
			type : "success",
			content : "关注成功"
		});
	}

	function toast2(obj) {
		javaex.toast(obj, {
			type : "error",
			content : "不能关注自己"
		});
	}
</script>

参数说明
参数 默认值 说明 版本
content 弹出层内容
type success success:成功。 error:失败
live 2000 提示存在时间，单位:毫秒 （仅在type为 success 或 error 时有效）
offsetTop 0 上偏移量
