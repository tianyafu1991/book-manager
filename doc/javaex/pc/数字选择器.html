数字选择器

代码
<!--可以直接给 input 设置值，也可以在组件中设置-->
<input id="numChose" class="javaex-numChose" value="1" readonly />

<script>
	javaex.numChose({
		id : "numChose",
		decrCallback : function(rtn) {
			console.log(rtn);
		},
		incrCallback : function(rtn) {
			console.log(rtn);
		}
	});
<script>
参数说明
参数 默认值 说明 版本
id 数字选择器组件的id
initValue 初始化显示的数字，也可以直接给 input 设置值
step 1 步长
minValue 1 最小值
maxValue 100 最大值
decrCallback 递减回调函数
incrCallback 递增回调函数
