标签

该组件没有用回调函数返回标签数组，而是采用input来存储数据，数据格式为： xxx,xxxx

是为了迎合 ajax 序列化提交

标签演示

最多添加6个标签

代码
<div class="javaex-tagbox"></div>
<input type="hidden" id="tag" name="tag" value="" />
<p class="javaex-hint">最多添加6个标签</p>

<script>
	javaex.tag({
		id : "tag"
	});
</script>
参数说明
参数 默认值 说明 版本
id 记录最终结果的input的id。格式为 xxx,xxxx
maxNum 6 标签数量上限
tags 标签回显，数组或list
