高级弹出层（窗口）

内嵌页面或网址或请求地址




代码
<script>
	$("#demo1").click(function() {
		javaex.dialog({
			type : "window",
			id : "test",	// 指定id，仅当页面存在多个弹出层，需要关闭指定弹出层时才使用
			url : "https://www.baidu.com/",	// 页面地址或网址或请求地址
			width : "800",	// 弹出层宽度
			height : "600",	// 弹出层高度
			hasBackground : true,
			close : function() {
				console.log("右上角点击X关闭时的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});
	
	$("#demo2").click(function() {
		javaex.dialog({
			type : "dialog",
			id : "test",	// 指定id，仅当页面存在多个弹出层，需要关闭指定弹出层时才使用
			url : "https://www.baidu.com/",	// 页面地址或网址或请求地址
			width : "800",	// 弹出层宽度
			height : "600",	// 弹出层高度
			close : function() {
				console.log("右上角点击X关闭时的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});

	$("#demo3").click(function() {
		javaex.dialog({
			type : "dialog",
			title : "添加",
			id : "test",	// 指定id，仅当页面存在多个弹出层，需要关闭指定弹出层时才使用
			url : "https://www.baidu.com/",	// 页面地址或网址或请求地址
			width : "800",	// 弹出层宽度
			height : "600",	// 弹出层高度
			close : function() {
				console.log("右上角点击X关闭时的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});

	function callback() {
		alert("回调函数");
	}
</script>
参数说明
参数 默认值 说明 版本
type window或dialog
mask true 是否使用遮罩。如果你想无限弹窗，必须将mask设置为false
id 指定id，仅在当页面存在多个弹出层，需要关闭指定弹出层时才使用。配合parent.javaex.close(id);使用
title 温馨提示 弹出层标题
url 页面地址或网址或请求地址
width 300 弹出层的宽度
height 弹出层的高度
scrolling yes 是否允许有滚动条。yes：有；no：没有
hasBackground false 是否有预加载图片。该图片不会消失，除非你的内容可以覆盖整个弹出层，例如嵌入播放器，否则不要设置为true
isClickMaskClose false 是否允许点击遮罩关闭弹窗
offsetLeft 0 偏移左边多少像素
offsetTop 0 偏移上边多少像素
close 右上角点击X关闭时的回调
callback 打开弹窗后的回调函数
callback : function() {
	console.log("弹窗已打开");
}
