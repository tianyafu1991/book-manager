页面预加载

页面渲染太慢时，可以使用该预加载提示作为友情提醒，页面加载完毕后，自动消失

自动，样式如下（当页面的DOM节点加载完毕后会自动消失，360等垃圾浏览器可能不支持）
正在读取数据，请稍候...
代码
<script>
	javaex.loading();
</script>
手动（全局），样式同上（适用于动态表单页）
代码
<script>
// 开启
javaex.loading({
	mode : "manual",
	type : "open"
});

// 关闭
javaex.loading({
	mode : "manual",
	type : "close"
});
</script>
手动（局部），样式如下（适用于ajax加载表格数据）
代码
<script>
// 开启
javaex.loading({
	mode : "manual",
	containerId : "table",
	type : "open"
});

// 关闭
javaex.loading({
	mode : "manual",
	containerId : "table",
	type : "close"
});
</script>
参数说明（使用属性默认值时，可以缺省该属性）
参数 默认值 说明 版本
id 提示的id，用于页面存在多个时，关闭指定的某一个
containerId 指定区域的id，如果不填，则加载提示会作用于整个页面
type 手动开启（open）或关闭（close）
mode 为空时表示调用自动的。manual 表示手动的
content 正在读取数据，请稍候... 加载提示内容（不适用于局部）
