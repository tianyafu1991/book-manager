针对复选框的 listen="listen-x" 属性全选。如果你想监听例如至少勾选一个复选框，才解除按钮的禁止点击状态，请调用如下方法，并将你的逻辑写在回调函数里
// 监听复选框选中事件
javaex.listenCheckbox({
	callback : function() {
		// 把你的逻辑写在这里
	}
});

伪复选框（一定要在外面包一层label）
代码
<label class="javaex-btn"><input type="checkbox" class="javaex-fake" name="checkbox3" value="1"/>B</label>
<label class="javaex-btn"><input type="checkbox" class="javaex-fake" name="checkbox3" value="2"/>奥黛丽</label>
<label class="javaex-btn"><input type="checkbox" class="javaex-fake" name="checkbox3" value="3"/>艾拉公主</label>
<label class="javaex-btn"><input type="checkbox" class="javaex-fake" name="checkbox3" value="4"/>玛丽</label>

<script>
	javaex.fakeCheckbox({
		name : "checkbox3",
		uncheckClass : "wathet", // 未选中时添加的class
		checkedClass : "blue" 	// 选中时添加的class
	});
</script>
参数说明
参数 默认值 说明 版本
name checkbox的name属性
uncheckClass 未选中时往label标签添加的class
checkedClass 选中时往label标签添加的class
